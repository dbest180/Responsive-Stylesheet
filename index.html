Colors!

My Responsive Design Project

Hi, I am DBest. This is my submitted CSS Homework Page. As you can see, I've created a responsive website that displays beautiful colors in three distinct views depending on your screen size!

  • On tablet screens (772px and above), the layout changes so the color divs are 45% of the viewport width, displaying two colors per row. The images also have a maximum height of 30% of the viewport height to keep everything properly sized.
  • On large screens (998px and above), the layout shifts again so the divs are 30% of the viewport width, allowing three colors to appear side by side.

I used media queries to make these layout changes trigger automatically at different viewport sizes. The divs use inline-block display to sit next to each other, and the width properties ensure they wrap to the next line when needed. Pretty cool, right?
